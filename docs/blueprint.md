# **App Name**: Piyush Joshi - Full Stack Portfolio

## Core Features:

- Hero Section: Visually appealing introduction with headline, sub-headline, and clear CTAs.
- About Me: Concise bio highlighting skills and aspirations, stored in Firestore.
- Skills Showcase: Categorized display of technical skills in development, databases and platforms.
- Projects Portfolio: Visually rich project cards with descriptions, tech stack, links to live demos and GitHub repositories.
- Experience & Achievements: Showcase professional experiences and notable achievements.
- Technical Blog: Blog that enables the user to upload technical blogs. Summarize the main idea and generate keywords of the uploaded article.
- Contact Form: Contact form that validates user inputs and submits the form to Firebase. Sends email notifications to the portfolio owner.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to convey professionalism and trust.
- Background color: Light gray (#F5F5F5) for a clean and modern backdrop.
- Accent color: Teal (#008080) for highlights and calls to action to bring a fresh and tech-forward vibe.
- Body and headline font: 'Inter', a grotesque-style sans-serif, to provide a modern and readable experience across all devices.
- Code font: 'Source Code Pro' for displaying code snippets in blog posts or project descriptions.
- Use simple, professional icons for skills, social media links (GitHub, LinkedIn, LeetCode), and navigation.
- Subtle animations and transitions on page load, hover states, and scrolling to enhance user engagement without being distracting.
